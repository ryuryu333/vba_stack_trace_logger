## コードのエクスポート・インポート
```bash
clone git@github.com:ryuryu333/vba_stack_trace_logger.git
cd vba_stack_trace_logger
# コードのエクスポート
cscript vbac.wsf decombine
# コードのインポート
cscript vbac.wsf combine
```
